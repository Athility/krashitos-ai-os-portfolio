# Design Style

Use modern UI trends:

soft gradients  
minimal color palette  
large typography  
clean spacing  

Avoid clutter.

Whitespace improves readability.

---

# Component Design

Design reusable components.

Examples:

feature cards  
pricing tables  
testimonials  
buttons  

Each component should include:

hover states  
focus states  
accessible interactions

---

# Visual Elements

Encourage use of:

illustrations  
subtle background animations  
gradient accents  

Avoid overly complex visuals.

Simplicity improves conversion.

# Responsive Design

Ensure layouts work on:

desktop  
tablet  
mobile  

Use flexible grids and breakpoints.

# Performance

Landing pages must load quickly.

Use:

optimized images

minimal JS bundles

modern
